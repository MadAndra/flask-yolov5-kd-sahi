Tentang Web Deteksi Objek Kecil

Aplikasi berbasis web ini memungkinkan pengguna untuk mengunggah gambar atau video untuk mendeteksi objek menggunakan model deep learning. Aplikasi ini mendemonstrasikan kemampuan model YOLOv5 yang telah dilatih khusus untuk mendeteksi dan mengklasifikasi objek dari citra drone secara real-time.

Fitur Utama:

  • Deteksi objek kecil pada gambar dengan tiga model berbeda secara real-time
  • Deteksi objek kecil pada video dengan tiga model berbeda secara real-time
  • Download hasil deteksi

Teknologi yang Digunakan:

Aplikasi berbasis web ini memanfaatkan teknologi deep learning dengan model YOLOv5 yang dikombinasikan dengan teknik Knowledge Distillation dan SAHI (Slicing Aided Hyper Inference ) untuk meningkatkan akurasi deteksi.

*note : hanya gunakan input berupa gambar atau video hasil tangkapan udara / citra drone untuk hasil yang lebih optimal
